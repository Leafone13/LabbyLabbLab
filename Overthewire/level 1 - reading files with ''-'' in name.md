The solution is something I somehow remembered by heart. Which is very cool byitself, haha

First we need to go to our specific folder. Since we're bandit1 - I tried to read what's inside bandit 1 by doing
```
cd bandit 1
ls
```
The file '' - '' was there, which I was looking for. 
To read what's inside I had to write down cat in a specific way

```
cat ./-
```

